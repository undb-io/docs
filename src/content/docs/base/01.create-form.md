A table is a core feature that provides users with a simple yet powerful way to organize and display data, such as for project management, data analysis, survey statistics, and scheduling. Users can customize the style and functionality of tables to meet various business needs.
。

# Create a table

In your Base panel, you will see a 'Create New Table' button. Click 'Create New Table' to create your first table.

![create table](/public/imgs/create-table.png)

# Just start typing

Let's start from scratch.
You can click in the header area to start typing the name of the table. For example, you can create a table named 'Team' to manage information about team members. When creating a table, the system provides a default String field, which you can rename to 'Name'.


![create table by name](/public/gif/creat-table-by-name.gif)

In addition to the default String field, you can add other fields according to your needs, such as an Attachment field for uploading photos of team members.

![add attachment field](/public/gif/add-attachment-field.gif)

# Select,move,duplicate, and delete field

You can select and modify any field, move their positions, or duplicate a field.

![move and duplicate](/public/gif/move-and-duplicate.gif)

To delete a field:

- Click on the dropdown button at the right of the field header to open the field operations menu.
- Click on 'Delete Field' in the menu options.

![delete field](/public/gif/delete-field.gif)

# System-created fields

When creating a table, the system automatically creates some fields which are initially hidden. You can choose to show these hidden fields to display them on the interface.


![show hide field](/public/gif/show-hide-field.gif)
