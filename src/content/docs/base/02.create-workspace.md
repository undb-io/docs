# Workspace

A "workspace" is a virtual environment where users create, design, and manage applications or projects. This interface is intuitive, allowing users to build their applications through drag-and-drop, configuration, and settings.

## Create workspace

# Base

"Base" refers to a foundational data model within an application used for storing and managing data. This includes defining data tables, fields, relationships, etc., providing the foundation for data storage and management in the application. Users can build their applications on this base, which may include predefined layouts, functional modules, and styles, serving as a quick starting point for new projects.

## Create Base

After registering an account and accessing the workspace homepage, if you are a new user with no existing data, the interface will display 'You have no bases' in the center. You can then click on 'Create New Base' in the center or in the left sidebar to initiate the creation of a new Base.

![create base by name](/public/imgs/create-base-by-name.png)
